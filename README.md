# hexo-filter-fenced-code

Extend syntax for the native fenced code block.

## The Native Syntax

```
```javascript
var foo = "bar";
`` `
```

## Highlight and Insert Code

```
````javascript
var foo = "bar";
``` `
```

## Just Insert Code

```
`````javascript
var foo = "bar";
```` `
```

**Note: Only JavaScript, CSS and HTML code can be insert to a post.**
